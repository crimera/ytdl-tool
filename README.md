# ytdl-tool
A youtube-dl script
# Install
`curl https://raw.githubusercontent.com/crimera/ytdl-tool/main/ytdl -o  ~/.local/bin/ytdl`

Optional - install termux-open-url

`mkdir ~/bin; curl https://raw.githubusercontent.com/crimera/ytdl-tool/main/ytdl.sh -o ~/bin/termux-url-opener; chmod +x ~/bin/termux-url-opener`
